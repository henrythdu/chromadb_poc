# ChromaDB POC - Project Overview

## Purpose
Greenfield proof-of-concept project for ChromaDB vector database.

## Tech Stack
- **Language**: Python 3
- **Vector Database**: ChromaDB (to be added)
- **Testing**: pytest
- **Code Quality**: ruff (linter + formatter)
- **Issue Tracking**: bd (beads) with git sync
- **Semantic Code Operations**: Serena MCP server
